# Cursor_clone_using_html_css


A sleek, high-fidelity recreation of the Cursor landing page. This project focuses on mastering modern web design patterns, particularly the "dark-mode" developer tool aesthetic.
https://cursor-clone-using-html-css.vercel.app/

## üöÄ Sections Recreated

This project focuses on the core landing page components that define Cursor's unique look:

* **Global Navigation:** High-blur translucent header with navigation links and the "Download" button.
* **Hero Section:** The centerpiece featuring the "The AI Code Editor" headline, primary CTA buttons, and the distinctive glowing background gradients.
* **Editor Preview:** A styled container mimicking the Cursor IDE interface, showcasing the code-editing UI and AI chat panels.
* **Feature Bento Grid:** A responsive grid layout highlighting specific features like "Tab to Tab," "Command K," and "Composer."
* **Interactive Hover States:** Custom CSS transitions on buttons and feature cards to mimic the premium feel of the original site.
